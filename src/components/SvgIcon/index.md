# SvgIcon

## 代码演示

### 基础

```tsx
// export default () => <SvgIcon svgName={'arrow'} nodeAttribute={{ fill: '#fff' }} />;
```

## API

| 属性          | 说明         | 类型                 | 默认值 |
| ------------- | ------------ | -------------------- | ------ |
| children      | 必填         | `React.ReactElement` |        |
| nodeAttribute | svg 标签属性 | `Record<string,any>` |        |
| style         | 样式         | `CSSProperties`      |        |
